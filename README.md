# Threat-Analysis-Project

I had the opportunity to work on shodan in order to look up several hosts of my choice in order to see the CVEs associated with the website. While searching for hosts, I get their IP and enter this command into my VM Ubuntu terminal as so while putting the IP where the x's are: curl https://internetdb.shodan.io/x.x.x.x. In doing so, I get some information in return including the CVEs that are located under the vulns: key. Some of the hosts I chose are in the screenshots.
